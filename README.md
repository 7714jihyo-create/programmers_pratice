# 📝 Programmers Coding Test Practice

프로그래머스(Programmers) 코딩 테스트 문제 풀이를 기록하는 저장소입니다.

---

## 🛠️ 연습 언어
  * **Language**: Java / C / Python / C#
* **Platform**: [Programmers](https://programmers.co.kr/)

---

### 🚀 Goal: Master the Core & Solidify the Basics
다양한 프로그래밍 언어를 접하며 생기는 문법적 혼동을 극복하고, 코드의 근간이 되는 언어별 기본기(Core Fundamentals)를 단단히 다지기 위한 공간입니다.

- 단순히 문법을 외우는 것을 넘어, 각 언어를 **깊이 있게 이해하는 것**,
- 사소한 문법적 실수(Syntax Error)를 줄이고, 어떤 언어를 마주하더라도 **흔들리지 않는 기본기**를 바탕으로 해결해나갈 수 있게 되는 것이 목표입니다
